Recognizer Object Documentation

Overview

The Recognizer object is responsible for listening to the microphone input and recognizing commands. It will implement the Subject interface from the Observer pattern, allowing it to notify the AI Assistant object when a command is detected.

This class implements the required methods for the Subject interface, as well as methods to start and stop listening to the microphone input. The private method _process_audio will be responsible for processing the audio input and detecting commands.

Methods

attach(observer: Observer)

Attaches an observer to the Recognizer's list of observers.

detach(observer: Observer)

Detaches an observer from the Recognizer's list of observers.

notify(command: str)

Notifies all attached observers with the detected command.

start_listening()

Starts listening to the microphone input.

stop_listening()

Stops listening to the microphone input.

_process_audio(audio)

Processes the audio input, detects commands, and notifies observers when a command is detected.
