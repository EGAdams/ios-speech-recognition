Using Design Patterns

As we incorporate the Factory Method, Observer, Strategy, and Decorator design patterns in our solution, the main AI Assistant object will be designed to take advantage of these patterns for flexibility and modularity.

AIAssistant

class AIAssistant:
    """Main AI Assistant object."""

def __init__(self, recognizer_factory, command_handler):
    """Initialize the AI Assistant."""
    self.recognizer = recognizer_factory.create_recognizer()
    self.command_handler = command_handler
    self.recognizer.attach(self)

def update(self, command):
    """Receive notifications from the CommandRecognizer when a command is detected."""
    self.command_handler.handle_command(command)

def start_listening(self):
    """Start listening for commands."""
    self.recognizer.start_listening()

def stop_listening(self):
    """Stop listening for commands."""
    self.recognizer.stop_listening()
